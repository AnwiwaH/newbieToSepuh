home back Tentu! Berikut adalah beberapa keybind yang dapat digunakan di EndeavourOS Xfce: Window Management: Alt + F4: Menutup jendela aktif. Alt + Tab: Beralih antara aplikasi yang sedang berjalan. Alt + F5: Memaksimalkan atau mengembalikan ukuran jendela. Alt + F9: Meminimalkan jendela aktif. Alt + F10: Maksimalkan jendela aktif. Alt + F11: Beralih ke mode layar penuh. Workspace Management: Ctrl + Alt + Arrow Keys: Beralih antara desktop yang berbeda. Ctrl + Alt + Shift + Arrow Keys: Memindahkan jendela aktif ke desktop lain. Application Launcher: Alt + F2: Membuka jendela "Run Program" untuk menjalankan aplikasi atau perintah. Super (Windows) + R: Membuka menu "Run Program". Super (Windows) + T: Membuka Terminal. File Management: Super (Windows) + E: Membuka file manager. Lock and Logout: Ctrl + Alt + Delete: Membuka jendela "Logout" untuk keluar dari sesi pengguna atau mengunci layar. Super (Windows) + L: Mengunci layar. Miscellaneous: Super (Windows) + D: Meminimalkan atau memulihkan semua jendela aktif. Alt + Print Screen: Mengambil tangkapan layar dari jendela aktif. Ini hanya beberapa contoh keybind yang umum digunakan di EndeavourOS Xfce. Anda juga dapat menyesuaikan keybind sesuai dengan preferensi Anda melalui pengaturan sistem.
